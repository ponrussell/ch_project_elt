# Современная Платформа Данных (Data Platform) на базе ClickHouse и Airflow

Самостоятельный учебный проект по созданию отказоустойчивого ELT-пайплайна для обработки аналитических логов (хитов) веб-сайта.

## 🛠️ Технологический стек
* **Оркестрация:** Apache Airflow 2.8.1 (LocalExecutor)
* **Хранилище данных:** ClickHouse 26.4 (Колоночная СУБД)
* **Объектное хранилище:** MinIO (S3-совместимое)
* **Трансформация данных:** dbt (Data Build Tool)
* **BI Визуализация:** Apache Superset 3.1.0
* **Развертывание:** Docker Compose

## 📐 Архитектура платформы (Паттерн Medallion)
1. **Слой Сбора (Inbound):** Логи кликов генерируются в формате Parquet и падают в бакет `bronze` хранилища MinIO.
2. **Слой Bronze (Raw данные):** Airflow с помощью `S3KeySensor` фиксирует файлы. ClickHouse через интеграционную функцию `s3()` напрямую скачивает данные в многопоточном режиме, минуя память Airflow. Производительность вставки: **1.4 млн строк за секунды**.
3. **Слой Silver (Валидация):** Данные очищаются от дубликатов, мусорных строк и обогащаются с помощью Materialized Views СУБД ClickHouse / моделей dbt.
4. **Слой Gold (Витрины):** Финальные бизнес-метрики агрегируются в DBT и выводятся на дашборды в Apache Superset.

## 🚀 Как запустить проект локально

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Запустите Docker-инфраструктуру:
   ```bash
   docker compose up -d
   ```
3. Откройте интерфейсы сервисов:
   * Airflow: `http://localhost:8080`
   * MinIO (S3): `http://localhost:9001`
   * Superset: `http://localhost:8088`
